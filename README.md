## Grok Chats

Chat with 3 different Grok personalities within a simple web Ui. This is meant as a quick tool
for generating stories using 3 different personalities.

** For the chat to work you must create a ".env" file and add a Grok AI 'API_KEY' **
** Import the required python imports aiohttp, dotenv, and xai_sdk**

1. Run python app.py
2. Open a web browser and go to http://localhost:8080
3. Chat with the peronalities.
